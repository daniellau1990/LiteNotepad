# LiteEdit 版本历史

## v0.1.1 (2026-04-16)

### 新增功能
- 状态栏更新指示器完整实现（8个任务完成）
  - 光标位置指示器（防抖更新）
  - 大文件模式指示器（>5MB检测）
  - 文件编码检测（支持UTF-8/UTF-16 BOM）
  - 行结束符指示器（CRLF/LF/CR/Mixed）
  - 自动保存状态指示器（60秒周期）
  - 文件监视状态指示器（外部修改检测）

### 修复问题
- 修复 StatusBar 组件接口不匹配问题
  - 更新 Props 接口支持 autoSaveState 和 fileWatchState
  - 移除废弃的 isAutoSaving 和 isFileWatched 属性
  - 统一 lineEndings 类型为 string 以兼容编码检测结果

### 技术改进
- 完整的 TDD 测试套件（16个测试文件）
- 前端架构优化：工具函数模块化、自定义 Hook 封装
- 后端 Tauri 命令完善（文件大小、修改时间、分块读取）

### 已知问题
- 依赖安装可能因网络问题失败
- 自动保存的实际文件写入逻辑待集成
- 大文件流式读取 UI 待优化
- 测试环境需要完整依赖安装

## v0.1.0 (2026-04-15)

### 初始版本
- 项目基础架构搭建
- Tauri + React + CodeMirror 6 集成
- 基础编辑器组件
- 菜单栏和状态栏框架
- 文件打开/保存基础功能
